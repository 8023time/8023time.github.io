字符串

一、字符串的基本定义与特性

1. 定义方式

  • 双引号："hello"
  • 单引号：'hello'
  • 反引号（模板字符串）：hello

2. 字符串的不可变性

  • 字符串是不可变类型，一旦创建就无法更改其内容，只能生成新的字符串。

三、字符串方法

1. 字符串的提取方法

  • **slice(start, end)**：提取指定范围的子字符串（不包括 end）。

  • **substring(start, end)**：类似 slice，但不支持负数索引。

  • **substr(start, length)**：从指定位置提取指定长度的子字符串（已废弃）。

2. 字符串的查找方法

  • **indexOf(substring, start)**：返回子字符串的首次出现索引，没有则返回 -1。

  • **lastIndexOf(substring)**：返回子字符串的最后一次出现索引。

  • **includes(substring)**：检查字符串是否包含子字符串。

  • **startsWith(substring)**：检查字符串是否以子字符串开头。

  • **endsWith(substring)**：检查字符串是否以子字符串结尾。

3. 字符串的修改方法

  • toUpperCase() 和 **toLowerCase()**：将字符串转换为大写或小写。

  • **trim()**：去除字符串首尾的空格。

  • **replace(search, replacement)**：替换匹配的子字符串。

  • **replaceAll(search, replacement)**：替换所有匹配的子字符串。

4. 字符串的分割和连接

  • **split(separator)**：将字符串按照指定分隔符分割为数组。

  • **concat(...strings)**：连接多个字符串。

5. 字符串的填充

  • **padStart(targetLength, padString)**：在字符串开头填充。

  • **padEnd(targetLength, padString)**：在字符串末尾填充。

6. 字符串的比较

  • 字符串可以通过 **>、<**、**>=**等运算符进行字典序比较。

7. 字符串的重复

  • repeat(n)：重复字符串 n次。

四、模板字符串（Template Literals）

1. 基本语法

  • 使用反引号定义，支持多行字符串和嵌入变量。

2. 表达式嵌入

  • 模板字符串可以嵌入任意表达式。

3. 标签模板

  • 使用函数对模板字符串进行处理。

五、字符串与其他类型的转换

1. 转为字符串

  • 使用 String() 函数或 .toString() 方法。

2. 转为数字

  • 使用 Number() 函数或 parseInt()/parseFloat()

六、字符串的迭代

1. for…of 循环

  • 可以使用 for…of 遍历字符串的每个字符。

2. 字符串拆分为数组

  • 使用 Array.from() 或 split()

七、常见字符串编码与解码

1. encodeURI 和 decodeURI

  • 编码和解码完整的 URI。

2. encodeURIComponent 和 decodeURIComponent

  • 编码和解码 URI 的组成部分。
